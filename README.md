# Wolverine VIP App

The ultimate "Pit Wall" experience for University of Michigan Season Ticket Holders.

## Features
*   **The Pit Wall (Home):** Real-time countdowns, Bento-grid layout, and live stats.
*   **Seat Command (Ticket):** Interactive 3D Season Pass with "Flip to Scan" functionality.
*   **Legacy (Renewal):** Gamified loyalty hub driving renewal conversions.

## Tech Stack
*   **Framework:** React Native (Expo SDK 50)
*   **Styling:** "Athletic Luxury" Theme (Custom Constants + StyleSheet)
*   **Animations:** `react-native-reanimated` + `moti`
*   **Icons:** `lucide-react-native`
*   **Effects:** `expo-blur` (Glassmorphism), `expo-linear-gradient`

## How to Run

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Start the Server:**
    ```bash
    npx expo start
    ```

3.  **Run on Device:**
    *   Scan the QR code with the Expo Go app on iOS or Android.
    *   Press `i` to run in iOS Simulator (if available).
    *   Press `a` to run in Android Emulator (if available).

## Design Philosophy
Inspired by the utility of F1 "Box Box" telemetry screens, adapted for the prestige of Michigan Athletics.
